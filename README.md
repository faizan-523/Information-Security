# 🔐 Information Security – Cipher Techniques (Python)

## 📖 Overview

This repository contains implementations of **classical cipher techniques using Python**.
The purpose of this project is to demonstrate how basic encryption and decryption algorithms work in Information Security.

These cipher techniques are commonly studied in **cryptography and cybersecurity fundamentals**.

---

## 🎯 Objective

The objective of this repository is:

* To understand the basics of **cryptography**
* To implement **cipher techniques using Python**
* To demonstrate **encryption and decryption processes**
* To practice programming concepts related to security

---

## 🔑 Cipher Techniques Implemented

This repository may include implementations of the following cipher algorithms:

* Caesar Cipher
* Row Transposition Cipher
* Playfair Cipher
* Autokey Cipher
* Vigenère Cipher
* One Time Pad Cipher

Each program demonstrates how plaintext can be converted into ciphertext and then decrypted back to plaintext.
